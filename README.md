# react-counter-app

## 프로젝트 개요
React + Vite로 만든 다중 증감 버튼이 있는 카운터 앱

## 기능 목록
- 현재 숫자 값을 화면에 표시
- -100, -10, -1, +1, +10, +100 버튼으로 값 증감
- 버튼 클릭 시 실시간으로 숫자 업데이트
- 상태 관리로 React의 useState 사용

 ## 기술 스택
 - HTML
 - CSS
 - Javascript
 - React
 - Vite
